git push不再需要重复输入用户密码的方法

使用hexo写博客，每次hexo deploy都要输用户名和密码，真是太麻烦了。
终于找到方法可以不用再输密码了。

  • 首先添加环境变量

计算机—属性—高级系统设置—环境变量

  • 在用户文件夹C:\Users\"yourusername"下新建一个名为_netrc的文件
