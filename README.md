### Web

1. `npm run build`
2. `gulp web`

### Chrome
1. `gulp chrome`

### Firefox 
1. `gulp firefox`

### iOS
1. 用`XCode`打开`ios/mapp.xcodeproj`并点击`Run`按钮。
2. `react-native start`

### Android
1. 配置环境(http://react-native.cn/docs/android-setup.html#content)
2. `react-native run-android`

### Desktop
1. `npm run desktop`
